# OpenStack Hackspace

## What you need

This document assumes you have:

- Rackspace credentials
- A computer with
	* Internet access
	* bash or zsh shell
	* Python
	* Python virtualenv package
	* SSH client

Talk to somebody in the room if you don't have these.
